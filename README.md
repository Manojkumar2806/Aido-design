# Aido Design

A clean, static landing page concept for **Aido — The Screen-Free Homework Buddy**.

## Overview

This repository contains a single-page website built with plain HTML and CSS. The page presents a product-style experience with:

- Hero and promotional sections
- Feature/content cards
- Pricing section
- Specs section
- FAQ accordion
- Footer with links and email form

## Tech Stack

- **HTML5**
- **CSS3**
- Google Fonts (`Outfit`, `Plus Jakarta Sans`)

## Project Structure

- `index.html` — Main page markup
- `style.css` — Complete styling for layout, typography, cards, sections, and responsive behavior

## Run Locally

No build tools are required.

1. Clone this repository
2. Open `index.html` in your browser

Or use a local server (optional), for example:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Notes

- The design uses externally hosted images and Google Fonts.
- This is a static frontend-only project (no backend or JavaScript logic).
